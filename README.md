# Blooket-H(acker)

Works for hacking blooket.

## Instructions:
1. Open `gui.js`
2. click on the text
3. Press `Ctrl` `a`
4. Press `Ctrl` `c`
5. Go to blooket
6. Press `Ctrl` `Shift` `j` and paste into the console.
  
   *You may need to allow pasting, so type: `allow pasting`*
   
9. Press `ok` on the popup and close the console.
10. Ready to play!
